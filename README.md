# Gameboy Adventure

Um jogo de plataforma feito em Python com Pygame Zero.

## Como Jogar
- **Setas Esquerda/Direita:** Mover
- **Seta Cima:** Pular
- **Objetivo:** Pule na cabeça dos inimigos para eliminá-los e chegue ao fim!

## Como Rodar
1. Instale as dependências:
   pip install -r requirements.txt

2. Rode o jogo:
   pgzrun main.py
